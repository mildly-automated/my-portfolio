Featured Project

A booking system that runs itself

Cal.com + Stripe Booking System

When I started taking on paid 1:1 work through Mildlyautomated, I needed a way to handle bookings and payments without hiring a developer or adding tools I didn't need yet.

I built a booking system using Cal.com connected directly to Stripe, synced with my existing Google Calendar and hosted on Netlify, so people can book and pay for a consultation or resume review in a single step.

It's a small build, but it's the backbone of how I take on client work now, proof that you don't need a complicated stack to start monetizing what you know.
